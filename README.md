# Project Theremin

A theremin is an early electronic musical instrument famous for being the only instrument played without physical contact. The musician controls the eerie, vocal-like sound by moving their hands through the air near two metal antennas, which alters the frequency of electromagnetic fields. (Wikipedia)

I'm working on this... I wish I had more time to work on my own projects.

For my investigation, I wrote a sketch in Javascript: https://blog.artaxia.com/theremin/play.html
I have used a framework for this, ml5.js. My knowledge of Javascript is quite limited.

The next step is creating a application that will run in my Raspberry Pi. The hand movements will be captured by a sensor.
